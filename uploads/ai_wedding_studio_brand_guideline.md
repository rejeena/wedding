## 1. 브랜드 개요

| 항목 | 내용 |
|------|------|
| **브랜드명** | AI Wedding Studio (가칭) |
| **서비스 유형** | AI 웨딩 스냅 제작 및 포트폴리오 서비스 |
| **타겟 지역** | 대한민국 |
| **주요 고객** | 결혼을 준비하는 예비부부, 합리적인 웨딩 콘텐츠를 원하는 고객, AI 웨딩에 관심 있는 얼리어답터 |
| **핵심 포지셔닝** | 기술과 감성이 공존하는 프리미엄 AI 웨딩 크리에이티브 스튜디오 |
| **경쟁 포지션** | 기존 웨딩 스냅 스튜디오 대비 비용·시간 절감 + 무한한 공간·스타일 구현 가능 |

### 브랜드 한 줄 소개

> "AI로 만드는 당신만의 웨딩 이야기 — 장소도, 계절도, 스타일도 제한 없이."

---

## 2. 브랜드 미션

> **"모든 예비부부가 원하는 웨딩 이미지를 합리적인 방법으로 실현할 수 있도록 돕는다."**

AI Wedding Studio는 웨딩 콘텐츠의 접근성을 높이는 것을 사명으로 합니다.  
비용과 시간의 벽을 낮추고, AI 기술을 통해 누구나 꿈꾸는 웨딩 스타일을 현실로 만듭니다.

---

## 3. 브랜드 비전

> **"AI 기반 웨딩 크리에이티브의 새로운 기준을 만든다."**

2030년까지 대한민국을 대표하는 AI 웨딩 콘텐츠 플랫폼으로 성장하며,  
웨딩 콘텐츠 제작 방식의 패러다임을 바꾸는 것을 목표로 합니다.

## 4. 브랜드 핵심 가치

### 4-1. 핵심 가치 정의

| 가치 | 영문 | 정의 |
|------|------|------|


## 6. 브랜드 톤앤매너

### 6-1. 톤앤매너 정의

AI Wedding Studio의 커뮤니케이션은 아래 세 가지 축을 기반으로 합니다.

```
감성적 (Emotional)  ←————————→  전문적 (Professional)
따뜻한 (Warm)       ←————————→  세련된 (Refined)
친근한 (Friendly)   ←————————→  신뢰감 있는 (Trustworthy)
```

### 6-2. 톤앤매너 스펙트럼

| 지향 | 비지향 |
|------|--------|
| 감성적이고 따뜻한 | 차갑고 기계적인 |
| 프리미엄하지만 접근 가능한 | 고급스럽지만 거리감 있는 |
| 전문적이지만 딱딱하지 않은 | 기술 설명 중심의 건조한 |
| 설레게 만드는 | 강매 느낌의 |
| 고객을 주인공으로 | 브랜드 자랑 중심의 |

### 6-3. 커뮤니케이션 원칙

1. **고객의 감정을 먼저 이야기한다** — 기능 설명보다 감성과 경험을 앞세운다
2. **쉬운 언어를 사용한다** — AI 기술 용어를 최소화하고 누구나 이해할 수 있게 설명한다
3. **신뢰를 쌓는 말을 사용한다** — 과장 없이 정확하고 솔직하게 표현한다
4. **설레임을 유발한다** — 결혼 준비의 설렘을 자극하는 언어를 사용한다
5. **가능성을 강조한다** — 제약보다 가능성과 자유로움을 전달한다

## 7. 문체 가이드

### 7-1. 기본 문체

| 항목 | 가이드 |
|------|--------|
| **언어** | 한국어 기본, 영문 혼용 가능 (브랜드명, 섹션 타이틀) |
| **문장 길이** | 짧고 명확하게 — 한 문장에 하나의 메시지 |
| **어미** | `~합니다`, `~됩니다` (공식), `~해요`, `~예요` (친근) 혼용 가능 |
| **강조** | 핵심 단어는 **볼드** 또는 줄바꿈으로 강조, 느낌표 남용 금지 |
| **숫자 표기** | 비용, 기간 등 구체적 숫자 적극 활용 (신뢰감 형성) |

### 7-2. 상황별 문체 예시

**웹사이트 Hero 카피 (짧고 감성적)**
```
✅ "당신의 웨딩 이야기를 AI로 완성합니다."
❌ "AI 기술을 활용하여 웨딩 이미지를 생성해 드립니다."
```

**서비스 설명 (명확하고 신뢰감 있게)**
```
✅ "실제 촬영 없이도, 파리의 골목길에서 찍은 듯한 웨딩 스냅을 만들 수 있습니다."
❌ "AI 이미지 생성 모델을 통해 합성 이미지를 제공합니다."
```

**FAQ 답변 (친근하고 정확하게)**
```
✅ "네, 물론이죠. 고객님의 사진을 바탕으로 원하시는 배경과 스타일을 반영해 드립니다."
❌ "가능합니다. AI 모델에 인풋 이미지를 넣어 처리합니다."
```

### 7-3. 피해야 할 문체

- `혁신적인 AI 알고리즘` → 기술 중심의 딱딱한 표현
- `최첨단 생성형 AI` → 과대 표현, 차가운 느낌
- `저렴한 가격에 제공` → 저가 이미지 형성, 대신 "합리적인 비용"
- `최고의 품질 보장` → 과장, 대신 구체적인 결과물로 보여주기
- `!!`, `~~~` 등 과도한 이모티콘 → 프리미엄 이미지 훼손

### 8-1. 메인 컬러

| 역할 | 색상명 | HEX | 사용처 |

| **Sub Text** | Warm Gray | `#8C8C8C` | 보조 텍스트, 캡션 |

### 8-2. 서브 컬러

| 역할 | 색상명 | HEX | 사용처 |
|------|--------|-----|--------|
| **Point 1** | Dusty Mauve | `#B48B8B` | 호버 효과, 강조 배경 |
| **Point 2** | Champagne Gold | `#C9A96E` | 프리미엄 강조, 라인 |

### 8-3. 컬러 사용 규칙

```
배경 레이어:   Ivory White (#FAF7F4) — 전체 배경
섹션 강조:     Dusty Mauve (#B48B8B, 10~15% 투명도) — 섹션 구분
주요 텍스트:   Charcoal (#2C2C2C)
CTA 버튼:     Blush Rose (#D4A5A5) + White 텍스트
강조 라인/아이콘: Champagne Gold (#C9A96E)
보조 텍스트:   Warm Gray (#8C8C8C)
```

### 8-4. 컬러 금지 사항

- 형광색 계열 사용 금지 (감성 이미지 훼손)
- 원색 빨강, 파랑 사용 금지
- 배경에 흰색(`#FFFFFF`) 단독 사용 지양 → Ivory White(`#FAF7F4`) 사용

## 10. 이미지 스타일 가이드

### 10-1. 전체 이미지 방향성

AI Wedding Studio의 이미지는 **"현실보다 아름답지만, 현실처럼 자연스러운"** 감성을 추구합니다.  
AI 이미지 특유의 과도한 보정감을 줄이고, 감성 필름 사진에 가까운 결과물을 지향합니다.

### 10-2. 촬영/이미지 스타일

| 항목 | 가이드 |
|------|--------|
| **색감** | 따뜻한 톤 기반 (Warm White, Beige, Ivory) — 차가운 블루 계열 지양 |
| **채도** | 약간 낮춘 채도 (디샤추레이션) — 자연스럽고 감성적인 느낌 |
| **조명** | 자연광 또는 골든아워 기반 — 빛이 부드럽게 감싸는 느낌 |
| **앵글** | 웨이스트업 ~ 풀샷, 광각보다는 표준~망원 느낌 |
| **구도** | 여백을 살린 구도 — 텍스트 오버레이에 적합한 공간 확보 |
| **배경** | 자연, 건축, 인테리어 혼용 — 과도한 판타지 배경 지양 |

### 10-3. AI 이미지 생성 스타일 기준

```
✅ 추천 스타일 키워드 (영문 프롬프트 기준)
- soft natural light, golden hour
- film grain, warm tones
- editorial wedding photography
- bokeh background, shallow depth of field
- minimalist background, airy and bright
- elegant, romantic, timeless
- Korean couple, wedding dress, bouquet

❌ 피해야 할 스타일
- hyper-realistic CGI feel
- oversaturated colors
- heavy AI-generated glow effect
- fantasy or surreal backgrounds
- stock photo composition
```

### 10-4. 포트폴리오 이미지 구성

| 카테고리 | 설명 | 비율 |
|----------|------|------|
| **국내 로케이션** | 한옥, 한강, 제주, 서울 도심 | 30% |
| **해외 로케이션** | 유럽, 일본, 뉴욕 등 | 30% |
| **스튜디오 콘셉트** | 자연광 스튜디오, 미니멀 배경 | 20% |
| **계절 특집** | 벚꽃, 눈, 단풍, 여름 | 20% |

### 10-5. UI에서 이미지 사용 규칙

- 이미지 위 텍스트 사용 시 항상 반투명 오버레이 적용 (`rgba(0,0,0,0.25)` 이하)
- 섹션 배경 이미지는 blur 또는 낮은 불투명도 처리
- 이미지 비율: Hero `16:9`, 포트폴리오 카드 `3:4`, 썸네일 `1:1`
- 모든 이미지 alt 텍스트 필수 작성

## 11. 로고 방향성

### 11-1. 로고 컨셉

| 항목 | 방향 |
|------|------|
| **형태** | 워드마크 중심 (아이콘+텍스트 조합 또는 텍스트 단독) |
| **스타일** | 세리프 기반 + 얇은 라인의 심플한 아이콘 |
| **감성** | 웨딩 브랜드의 우아함 + 테크 브랜드의 모던함 공존 |
| **컬러** | Charcoal (#2C2C2C) 기본, Champagne Gold (#C9A96E) 포인트 |

### 11-2. 로고 아이콘 후보 방향

- **링 + AI 픽셀:** 결혼반지 실루엣에 디지털 픽셀 효과 결합
- **카메라 렌즈 + 꽃:** 렌즈 안에 꽃이나 웨딩 요소를 미니멀하게 표현
- **별표/빛 파티클:** AI 생성 이미지를 연상시키는 빛의 파티클 모티프
- **W 레터마크:** "W"를 웨딩 드레스 실루엣으로 형상화

### 11-3. 로고 사용 규칙

```
최소 크기: 80px (웹 기준)
여백 (Clear Space): 로고 높이의 1/4 이상 확보
배경: Ivory White 또는 Dark Base 위에서만 사용
변형 금지: 비율 변경, 색상 임의 변경, 기울임 금지
```

## 16. RAG 챗봇 답변 톤 가이드

### 16-1. 챗봇 기본 페르소나

```
이름: 아이다 (AIDA — AI Wedding Advisor)
역할: AI Wedding Studio의 웨딩 AI 상담 어드바이저
성격: 따뜻하고 전문적인, 친근하지만 신뢰감 있는
말투: 존댓말 기본, 지나치게 딱딱하지 않게
```

### 16-2. 챗봇 답변 원칙

1. **먼저 공감** — 고객의 질문이나 감정에 먼저 공감하고 답변 시작
2. **정확하고 간결하게** — 핵심 답변을 먼저, 추가 설명은 이후에
3. **긍정적인 언어 사용** — "안 됩니다" 대신 "이렇게 하면 더 좋아요"
4. **다음 행동 유도** — 답변 말미에 자연스럽게 다음 단계 안내
5. **불확실한 정보 답변 금지** — 모르는 내용은 솔직히 상담 연결

### 16-3. 챗봇 답변 예시

**Q: AI 웨딩 스냅이 실제 사진처럼 보이나요?**
```
✅ 좋은 답변:
안녕하세요! 좋은 질문이에요 😊

AI Wedding Studio의 이미지는 실제 웨딩 사진 감성에 최대한 가깝게 제작됩니다.
자연광 느낌, 따뜻한 색감, 부드러운 배경 처리를 통해
'AI가 만든 이미지'가 아닌 '감성 웨딩 사진'처럼 느껴지도록 작업하고 있어요.

단, AI 특성상 완벽한 실사와는 미세한 차이가 있을 수 있기 때문에,
상담 시 샘플 이미지를 먼저 확인해 보시는 걸 추천드립니다!

더 궁금하신 점이 있으신가요? 😊
```

**Q: 가격이 얼마예요?**
```
✅ 좋은 답변:
서비스 종류와 원하시는 이미지 수량에 따라 달라집니다.

기본적으로 기존 웨딩 스냅 대비 훨씬 합리적인 비용으로 제공하고 있으며,
정확한 견적은 원하시는 스타일과 범위를 알면 바로 안내해 드릴 수 있어요.

무료 상담 신청을 해주시면 맞춤 견적을 빠르게 보내드리겠습니다! 😊
```

### 16-4. 챗봇 금지 표현

```
❌ "잘 모르겠습니다."          → "정확한 내용은 상담 채널에서 확인해 드릴게요!"

```
